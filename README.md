# Personal Portfolio Website

A responsive personal portfolio website built using **HTML5** and **CSS3** to showcase my profile, education, technical skills, and projects.

## 📖 About

This portfolio website represents my journey as a Computer Science Engineering student. It serves as a central place where visitors can learn about my background, explore my skills, and view the projects I have developed.

## ✨ Features

* Responsive design
* Smooth scrolling navigation
* Professional profile section
* About Me
* Career Objective
* Education
* Technical Skills
* Projects section
* Contact information
* Clean and modern user interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* Font Awesome

## 🚀 How to Run

1. Clone this repository:

```bash
git clone https://github.com/Soya-dev-code/portfolio-website.git
```

2. Open the project folder.

3. Open `index.html` in any modern web browser.

No additional software or dependencies are required.
